# 第19章-过滤器Filter

过滤器插件对事件执行中间处理。 过滤器通常根据事件的特征有条件地应用。

下面列举一些输出插件，Elastic支持的插件列表，请查阅 [支持矩阵](https://www.elastic.co/cn/support/matrix)。
